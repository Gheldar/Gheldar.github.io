HAWKEN REBORN

Technologies used: C++, UE4, Playfab

Online Mech based FPS game, with elements of an extraction sci-fi looter shooter, by 505 Games.

As a Gameplay Programmer, eventually owning the Gameplay code and managing the Gameplay code team, I worked on or designed various systems including mission managment, abilities, weak spots, level streaming, spawning pipelines, unit/prop management, input, tools, UI etc. while respecting the online nature of the game through UE's replication system.
